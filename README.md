# FSD second task - Template layout

FSD задание номер 2 - вёрстка
  
## Clone repository
``` 
git clone https://github.com/Constantine-P/FSD-Frontend-2-task.git
```

## Installation
``` 
npm i
```

## Run development server
``` 
npm start
```

## Building
``` 
npm run build
```
